## 🔧 USAGE EXAMPLES

```bash
# Monitor mempool on Base
curl -X POST https://lb.drpc.live/base/Avibgvi26EjPsw76UtdwmsQzlkwrEmsR8b2u-uF7NYYO \
  -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"txpool_content","params":[]}' | jq '.result.pending | keys'

# Monitor mempool on Ethereum
curl -X POST https://1rpc.io/eth \
  -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"txpool_content","params":[]}' | jq '.result.pending | length'

# Get pending count on Avalanche
curl -X POST https://avalanche.drpc.org \
  -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"txpool_status","params":[]}'
```
